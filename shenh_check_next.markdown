### **请求接口**
/index.php?app=Kaoqin&m=KaoqinApi&a=shenh_check_next

### **请求方式**
get/post

### **请求参数**

| 参数名称  |必填|     说明      |
|------|-----|------|
| uid     | 是 |   员工id   |
| cid     | 是 |   公司id   |
| next_uid | 是 |   提交给下一个人审核的员工的id   |
| jilu_id | 是 |   该条审核记录的id   |


### **返回结果**
|字段       |值             |类型    |说明           |
| --------- |--------      |--------|--------       |
|status     |1/0/-1 |string |请求状态： -1被删除了 ； 0失败 ； 1成功   |
|info       |成功、失败        |string         |返回结果说明    |
|-----      |-----         |-----  |-----           |
