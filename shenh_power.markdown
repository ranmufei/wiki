### **请求接口**
/index.php?app=Kaoqin&m=KaoqinApi&a=shenh_power

### **请求方式**
get/post

### **请求参数**

| 参数名称  |必填|     说明      |
|------|-----|------|
| uid | 是 |   员工id   |
| cid     | 是 |   公司id   |

|字段       |值             |类型    |说明           |
| --------- |--------      |--------|--------       |

|data       | 打卡记录       |num       |返回数组    |
